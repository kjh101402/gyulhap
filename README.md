## 결합 그림을 난수로 생성시 결과
### 그림 생성을 10000번씩 총 10번한 결과 표다
![결과표](./image/testResultTable.png)

중복을 허용하고 했을 때 결이 5%정도 확률로 나왔으나 그림은 중복되면 안되기 때문에 다시 코드를 짰다

코드는 RandomGyulhapTest.java 에 올려놓았다.   
난수를 이용해 프로젝트를 진행하는 것으로 결정했다.
